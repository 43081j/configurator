# configurator

This monorepo holds the sources for the _configurator_ - a tool to generate configuration files for various developer tools given a meta-config.

## Usage

The configurator is currently hosted at [https://43081j.com/configurator/](https://43081j.com/configurator/).

## CLI (experimental)

A CLI exists in this monorepo which **is not yet published on npm**.

This CLI achieves the same as the UI, for example:

```bash
npx @43081j/configurator-cli --ui-framework angular --test-framework vitest
```
